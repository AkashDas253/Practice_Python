# Sorting Algorithms in Python

This project implements six classic sorting algorithms in Python, each in its own module with step-by-step verbose output support:

- Selection Sort (`selection_sort.py`)
- Insertion Sort (`insertion_sort.py`)
- Bubble Sort (`bubble_sort.py`)
- QuickSort (`quicksort.py`)
- MergeSort (`mergesort.py`)
- HeapSort (`heapsort.py`)

## Usage

### Interactive Sorting

Run the menu-driven script to choose an algorithm, input an array, and see step-by-step working:

```sh
python sorting_algorithms/main.py
```

### Automated Testing

Run all algorithms on multiple test cases to verify correctness:

```sh
python sorting_algorithms/test_sorting.py
```

## Features
- Each algorithm supports a `verbose` mode for step-by-step explanation.
- Easily add more algorithms by creating new modules and updating the menu/test scripts.
