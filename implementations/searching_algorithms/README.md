# Searching Algorithms

This module contains implementations of standard searching algorithms in Python, with a menu-driven main program and verbose step-by-step output option.

## Algorithms Implemented
- Linear Search
- Binary Search
- Jump Search
- Exponential Search

## Usage
Run `main.py` to interactively choose an algorithm, input an array and target, and optionally see verbose output of each step.

## Verbose Option
You can enable verbose mode in `main.py` to see detailed steps for each algorithm.

## Tests
Run `test_searching.py` to validate all algorithms against multiple test cases.
