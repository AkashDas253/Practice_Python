## Practice Python Calculators

This repository contains two calculator applications built with Python and Tkinter:

### 1. Standard Calculator
- Location: `calculator_gui/main.py`
- Features: Basic arithmetic operations (+, -, *, /), parentheses, decimal point, clear, backspace.
- Usage: Run the file to open a simple GUI calculator.

### 2. Scientific Calculator
- Location: `scientific_calculator/main.py`
- Features: All standard operations plus:
  - Square root (√)
  - Logarithm (log)
  - Trigonometric functions (sin, cos, tan)
  - Exponential (exp)
  - Power (^)
  - Constants (π, e)
  - Absolute value (abs)
- Usage: Run the file for a scientific calculator with advanced functions.

#### How to Run
1. Make sure you have Python installed.
2. Open a terminal in the project folder.
3. Run either calculator:
   - Standard: `python calculator_gui/main.py`
   - Scientific: `python sci_cal/main.py`
