# 2048 Game

A simple implementation of the classic 2048 puzzle game using Python and Tkinter.

## Features
- Playable GUI using Tkinter
- Score tracking
- Restart option when the game ends
- Smooth tile movement with arrow keys

## How to Play
1. Run `main.py` to start the game.
2. Use your keyboard arrow keys (Up, Down, Left, Right) to move the tiles.
3. Tiles with the same number merge when they touch, forming a new tile with their sum.
4. Your goal is to create a tile with the number 2048.
5. The game ends when no moves are possible. You can restart from the popup.


## Requirements
- Python 3.x
- Tkinter (usually included with Python)
- See `requirements.txt` for details.

To install requirements, run:
```
pip install -r requirements.txt
```

## How to Run
Open a terminal in this folder and run:

```
python main.py
```
