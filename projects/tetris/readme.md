# Tetris Game (Python, Pygame)

This project is a graphical Tetris game built using Python and the `pygame` library.

## Features
- Classic Tetris gameplay
- Colorful blocks and smooth controls
- Score tracking
- Game over and restart functionality

## Getting Started

### Prerequisites
- Python 3.x
- `pygame` library

### Setup
1. (Optional) Create and activate a virtual environment:
	```powershell
	python -m venv venv
	.\venv\Scripts\activate
	```
2. Install dependencies:
	```powershell
	pip install pygame
	```

### Running the Game
1. Save the Tetris game code in a file named `tetris.py` in this folder.
2. Run the game:
	```powershell
	python tetris.py
	```

## Controls
- Left/Right Arrow: Move block
- Up Arrow: Rotate block
- Down Arrow: Soft drop
- Space: Hard drop (if implemented)
- Q or Esc: Quit
