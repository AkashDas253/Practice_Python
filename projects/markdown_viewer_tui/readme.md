# Markdown Viewer TUI

A simple terminal-based Markdown viewer built with the Textual framework.

## Features
- View markdown files in the terminal with formatting
- Keyboard navigation (coming soon)

## Usage

1. Install dependencies:
   ```sh
   pip install textual rich
   ```
2. Run the app:
   ```sh
   python main.py path/to/yourfile.md
   ```

## Project Structure
- `main.py` — Main application entry point
- `readme.md` — Project documentation
